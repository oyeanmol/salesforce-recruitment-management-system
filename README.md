# Salesforce Recruitment Management System

## 📌 Overview

This project is a complete Recruitment Management System built on Salesforce. It manages job positions, candidates, job applications, and hiring workflows with automation and approval processes.

---

## 🚀 Features

### 🔹 Core Modules

* Positions Management
* Candidates Tracking
* Job Applications
* Job Postings
* Reviews System

---

### 🔹 Advanced Salesforce Features

* Record Types (Technical & Non-Technical)
* Page Layout Customization
* Dependent Picklists
* Validation Rules
* Flows (Automation)
* Approval Process (Manager Approval System)

---

### 🔹 Automation Implemented

* Auto close positions based on close date
* Reopen logic
* Approval-based status updates

---

### 🔹 Apex (Light Development)
* Trigger to prevent duplicate job applications
* Test class for validation and coverage

---

### 🔹 Dashboard & Reporting

* Recruitment insights dashboard
* Position tracking analytics

---

## 🛠️ Tech Stack

* Salesforce Admin
* Flows
* Approval Processes
* Reports & Dashboards

---
